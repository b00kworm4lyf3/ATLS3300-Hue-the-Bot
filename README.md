"HUE THE BOT"

Need a break from your work, but don’t want to use your phone? Hue is a robot desk friend that keeps you engaged while providing you company. Using a color sensor and an initiation button, he is able to scan the color of physical objects. With this reading, he is able to announce the hexcode on his belly screen and display that RGB color through his LED cheeks. He is also interactive and responds to movement with an accelerometer. If you pick him up or nudge him, he will react with facial animations, which are also affected by color outputs.
